# Changelog

All notable changes to this project will be documented in this file.

## [1.0.0] - 2026-05-04

### Added

- Initial design token package setup with Style Dictionary.
- Source token collection in `tokens.json`.
- Generated JavaScript token output in `build/js/tokens.js`.
- Generated React theme output in `semantic/react/theme.ts`.
- Base color tokens: black, white, grey, Tropical Teal, blue, yellow, purple, coklat, greynew, springgreen, darkcyan, skyaqua, abubutek, charcoal, and coffeebean.
- Basic localized text tokens for Indonesian and English.

### Changed

- Synced generated token artifacts with the latest source tokens.
- Updated project remote to `https://github.com/coolbuilds/Token-Design-Base`.
